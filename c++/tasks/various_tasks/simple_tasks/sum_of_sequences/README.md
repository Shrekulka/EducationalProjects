Данный код вычисляет и выводит на экран сумму чисел последовательности, где каждый следующий элемент получается путем 
добавления значения 'B' к предыдущему элементу ('A' + i * 'B'), для i от 0 до 'N'-1. Кроме того, выводится также сумма 
всех элементов последовательности (summSequence).

1. Запрашивает у пользователя три значения: действительное число 'A', действительное число 'B' и натуральное число 'N'.
2. Инициализирует переменные summ и summSequence со значением 0.
3. Запускает цикл, который будет выполняться от 0 до n-1.
4. В каждой итерации цикла вычисляет значение summ как сумму числа 'A' и произведения числа 'B' на значение i.
5. Добавляет значение summ к переменной summSequence.
6. Выводит значение summ на экран.
7. По завершении цикла выводит значение summSequence, которое является суммой всех вычисленных значений summ.
